## Image Guidelines

1. **Format**: Use JPG for photos, PNG for icons with transparency
2. **Optimization**: Compress images for web (use tools like TinyPNG)
3. **Naming**: Use lowercase with hyphens (kebab-case)
4. **Fallback**: If images are missing, the site will show emoji fallbacks
