# 🎨 Personal Portfolio & Content Management System

A modern, full-stack portfolio application with blog, projects showcase, and admin dashboard.

## 🚀 Quick Start

```bash
# Backend
cd backend
npm install
npm start

# Frontend (new terminal)
cd frontend
npm install
npm run dev
```

**Open:** http://localhost:5173  
**Admin:** http://localhost:5173/admin/login  
**Default Login:** admin@example.com / admin123

## ✨ Features

- 🎨 Modern, responsive design
- 📝 Blog with video support (YouTube, TikTok, Instagram, etc.)
- 💼 Projects showcase with status badges
- 🛠️ Skills organized by category
- 📧 Contact form with validation
- 🔐 Secure admin dashboard
- 🎭 Animated briefcase login page
- 💬 Testimonials slider
- 📱 Mobile-responsive navigation
- 🌙 Dark mode toggle

**Full list:** [docs/FEATURES_ADDED.md](docs/FEATURES_ADDED.md)

## 🛠️ Tech Stack

**Frontend:** React 18, Redux Toolkit, React Router, Vite  
**Backend:** Node.js, Express, MongoDB, JWT  
**Styling:** CSS-in-JS with responsive design
