# ⚡ Quick Embed Reference Card

## Copy-Paste This Into Your Blog Posts

---

## 📺 YouTube
```html
<iframe width="100%" height="400" 
  src="https://www.youtube.com/embed/VIDEO_ID" 
  frameborder="0" allowfullscreen>
</iframe>
```
**Get ID from:** `youtube.com/watch?v=VIDEO_ID`

---

## 🎵 TikTok
```html
<blockquote class="tiktok-embed" 
  cite="https://www.tiktok.com/@USER/video/VIDEO_ID" 
  data-video-id="VIDEO_ID" 
  style="max-width: 605px; margin: 0 auto;">
</blockquote>
<script async src="https://www.tiktok.com/embed.js"></script>
```
**Get ID from:** `tiktok.com/@USER/video/VIDEO_ID`

---

## 📸 Instagram
```html
<blockquote class="instagram-media" 
  data-instgrm-permalink="https://www.instagram.com/p/POST_CODE/" 
  data-instgrm-version="14" 
  style="margin: 0 auto;">
</blockquote>
<script async src="//www.instagram.com/embed.js"></script>
```
**Get Code from:** `instagram.com/p/POST_CODE/`

---

## 🐦 Twitter
```html
<blockquote class="twitter-tweet">
  <a href="https://twitter.com/USER/status/TWEET_ID"></a>
</blockquote>
<script async src="https://platform.twitter.com/widgets.js"></script>
```
**Get ID from:** `twitter.com/USER/status/TWEET_ID`

---

## 🎥 Vimeo
```html
<iframe 
  src="https://player.vimeo.com/video/VIDEO_ID" 
  width="100%" height="400" 
  frameborder="0" allowfullscreen>
</iframe>
```
**Get ID from:** `vimeo.com/VIDEO_ID`

## 🎯 Most Common: YouTube + TikTok

### For Long Tutorials (5+ min):
```markdown
# My Tutorial

<iframe width="100%" height="400" src="https://www.youtube.com/embed/YOUR_ID" frameborder="0" allowfullscreen></iframe>

## What You'll Learn
- Point 1
- Point 2
```

### For Quick Tips (< 1 min):
```markdown
# Quick Tip

<blockquote class="tiktok-embed" cite="https://www.tiktok.com/@user/video/123" data-video-id="123" style="max-width: 605px; margin: 0 auto;"></blockquote>
<script async src="https://www.tiktok.com/embed.js"></script>

## Explanation
...
```

## 💡 Pro Tips

1. **Always add written content** - Not everyone watches videos
2. **Test on mobile** - Make sure it works on phones
3. **Use 100% width** - Makes videos responsive
4. **Add fallback links** - In case embed fails

## ⚠️ Common Mistakes

❌ **Don't do this:**
```html
<iframe src="youtube.com/watch?v=123"></iframe>
```

✅ **Do this:**
```html
<iframe src="https://www.youtube.com/embed/123"></iframe>
```

---

❌ **Don't forget the script:**
```html
<blockquote class="tiktok-embed"></blockquote>
<!-- Missing script! -->
```

✅ **Include the script:**
```html
<blockquote class="tiktok-embed"></blockquote>
<script async src="https://www.tiktok.com/embed.js"></script>
```
