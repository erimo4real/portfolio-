## 🛠️ Tech Stack

**Frontend:**
- React 18
- Redux Toolkit
- React Router v6
- Vite
- Marked (Markdown parser)

**Backend:**
- Node.js
- Express
- MongoDB + Mongoose
- JWT Authentication
- Multer (File uploads)
- Bcrypt (Password hashing)

**Details:** [IMPLEMENTATION_SUMMARY.md](IMPLEMENTATION_SUMMARY.md)
